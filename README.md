# Radiant Rise Website

A production-ready web application for Radiant Rise, built with Next.js 16, TypeScript, Tailwind CSS, and Sanity.io.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS (High-Contrast Monochrome)
- **Icons**: Lucide React
- **CMS**: Sanity.io

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fradiant-rise%2Fwebsite)

The project is configured for one-click deployment on Vercel.

## Configuration

- **Colors**: Modified in `lib/tokens/colors.ts`
- **Typography**: Modified in `lib/tokens/typography.ts`
